CareCircle v4.5 — Weekly Care Packet

New:
- Added a Weekly Care Packet option in the Print Center.
- Weekly packets use the selected Summary Date as the week starting date and include 7 days.
- Updated the print button wording to "Print Care Packet".
- Updated Print Center helper text and version display.

Test:
1. Open Print Center.
2. Select "Print weekly packet".
3. Choose a summary date / week starting date.
4. Confirm the preview shows "Week of ... through ...".
5. Print or save as PDF.
